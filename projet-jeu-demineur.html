Projet Scolaire

Projet Jeu demineur en Java-Swing

GLPI

L'objectif du jeu est de révéler toutes les cases libres sans déclencher les mines, c'est-à-dire sans cliquer sur les cases qui les contiennent. Lorsqu'un joueur clique sur une case libre adjacente à au moins une mine, un chiffre apparaît, indiquant le nombre de mines situées dans les cases avoisinantes.
